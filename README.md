# pro-sjb26-archive

## 项目简介

本仓库用于归档与发布多个独立的 HTML 页面。这些页面可能包含工具、演示、实验性功能或信息展示等内容，旨在提供一种简单、轻量化的网页托管与分发方式。

仓库本身不针对任何特定域名或网站，所有页面均为独立构建，彼此无业务关联。使用者可根据需要直接通过 GitHub Pages 或本地浏览查看。

## 目录说明

```
/
├── index.html          # 仓库主页，列出所有归档页面入口
├── pages/              # 存放所有独立 HTML 页面
│   ├── page-1.html
│   ├── page-2.html
│   └── ...
├── assets/             # 公共资源文件（CSS、JS、图片等）
│   ├── css/
│   ├── js/
│   └── images/
└── README.md           # 本文件
```

- `pages/`：每个 HTML 文件是一个独立的归档页面，可单独访问。
- `assets/`：存放多个页面可能共用的样式、脚本或图片资源。
- `index.html`：提供页面导航列表，方便浏览所有归档内容。

## 页面归档说明

- 每个页面均为完整自包含的 HTML 文件，或依赖 `assets/` 下的公共资源。
- 页面内容可能不定期更新或移除，不保证长期稳定性。
- 归档页面仅供学习、参考或临时使用，不承担任何生产环境责任。
